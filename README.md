# Toolbox Monorepository
Code that can be reused among different projects

## 📦 Available Packages

- `@melledijkstra/api` - API client utilities
- `@melledijkstra/auth` - Authentication helpers (OAuth with arctic)
- `@melledijkstra/config` - Shared configuration
- `@melledijkstra/extension` - Browser extension utilities
- `@melledijkstra/storage` - Storage abstractions
- `@melledijkstra/toolbox` - General utilities
- `@melledijkstra/ui` - UI components (React/Svelte)
